##Ultimate Gallery plugin for OctoberCMS
Manage your images easily. This plugin based on [RJGallery](http://octobercms.com/plugin/raviraj-rjgallery).
I extended the album with some useful properties.

###Main features
* Managing albums
* Managing categories

###Extra features
* Multilanguage support to text fields.
* Available the wide range of widgets.

###Frontend widgets
* Images
* Albums
* Categories

###Backend widgets
* Albums
* Categories

###Frontend booking
1. Go to the __CMS > Pages__ page in Backend.
1. Added __Gallery__ components in Components panel.

###Available languages
* en - English
* hu - Magyar

###Installation
1. Go to the __Settings -> Updates__ page in Backend.
1. Click on the __Install Plugins__ button.
1. Type the __Ultimate Gallery__ in search field.

###Add widget
1. Go to the __Dashboard__ page in Backend.
1. Click on the __Add widget__ button.
1. __Select__ the current widget from the list.
